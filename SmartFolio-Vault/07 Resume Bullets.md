# Resume Bullets

## Project Title

SmartFolio: Agentic AI Portfolio and Stock Analysis Platform

## Strong Version

- Designed and deployed **SmartFolio**, an AI-assisted investment intelligence platform on Vercel combining investor onboarding, portfolio diagnostics, scenario modeling, and OpenVC-style stock analysis.

- Built an **A2A-style multi-agent architecture** with Profile, Risk, Portfolio, Stock Forecast, Backtest, Recommendation, Compliance, and Advisor agents for explainable financial analysis workflows.

- Implemented deterministic analytics for allocation gaps, concentration risk, risk scoring, forecast bands, backtest summaries, and portfolio-level stock impact analysis.

- Created an interactive dashboard with guided setup, **Skip to Demo** flow, portfolio workspace, OpenVC-style **Analyze Stock** terminal, AI advisor chat, agent topology, audit logs, research memory, scenario controls, and connected-app architecture.

- Architected the platform for production expansion with planned support for FastAPI services, typed Pydantic models, live market data APIs, broker and CSV imports, MCP tool servers, LLM routing, persistent memory, Postgres storage, health checks, provider fallback, and rate limiting.
