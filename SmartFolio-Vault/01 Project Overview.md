# Project Overview

## Name

SmartFolio

## Resume Title

SmartFolio: Agentic AI Portfolio and Stock Analysis Platform

## Product Summary

SmartFolio is a public-facing AI investment intelligence platform. It supports:

- guided investor onboarding
- portfolio diagnostics
- OpenVC-style stock analysis
- scenario simulation
- AI advisor interactions
- connected-app architecture

## Why This Project Matters

SmartFolio is stronger than a simple stock app because it connects individual stock analysis to the user's broader financial profile and portfolio.

Instead of only asking:

> Should I analyze this stock?

SmartFolio asks:

> Does this stock fit my portfolio, risk profile, and goals?

## Main Modules

- Guided Setup
- Portfolio Workspace
- Analyze Stock
- Scenario Lab
- AI Advisor
- Connections Hub
